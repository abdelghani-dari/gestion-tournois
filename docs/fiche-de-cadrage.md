# Fiche de Cadrage — Gestion Tournois

## 1. Intitulé du projet

Application de Gestion Sportive — Gestion Tournois

## 2. Contexte du projet

Dans le cadre du projet tutoré, notre équipe développe une application web de gestion sportive.

L'objectif est de permettre aux responsables sportifs de gérer facilement les saisons, championnats, tournois, équipes, joueurs, matchs, compositions, classements et statistiques.

## 3. Problématique

La gestion manuelle des compétitions sportives peut devenir difficile lorsque le nombre d'équipes, de joueurs et de matchs augmente.

Les responsables sportifs ont besoin d'un outil centralisé permettant de suivre les compétitions, planifier les matchs, enregistrer les résultats et générer automatiquement les classements.

## 4. Objectifs du projet

- Gérer les saisons sportives.
- Gérer les championnats.
- Gérer les tournois.
- Gérer les équipes.
- Gérer les joueurs.
- Planifier les matchs.
- Saisir les résultats.
- Gérer les compositions d'équipes.
- Générer les classements.
- Consulter les statistiques.

## 5. Périmètre du projet

### Inclus dans le projet

- Interface web frontend.
- API backend.
- Base de données PostgreSQL.
- Gestion CRUD des principales entités.
- Classement automatique.
- Statistiques simples.
- Déploiement local avec Docker Compose.
- Documentation technique.

### Non inclus dans la première version

- Application mobile.
- Paiement en ligne.
- Notifications en temps réel.
- Gestion avancée des arbitres.
- Système complexe de billetterie.

## 6. Utilisateurs cibles

- Responsables sportifs.
- Administrateurs de compétitions.
- Membres d'organisation sportive.
- Utilisateurs simples souhaitant consulter les résultats et classements.

## 7. Technologies utilisées

| Partie | Technologie |
|---|---|
| Backend | Laravel |
| Frontend | React |
| Base de données | PostgreSQL |
| API | REST API |
| Déploiement local | Docker + Docker Compose |
| Versioning | Git + GitHub |
| Conception | UML |

## 8. Résultat attendu

À la fin du projet, l'application devra permettre la gestion complète d'une compétition sportive, depuis la création des équipes jusqu'à la génération des classements et statistiques.
